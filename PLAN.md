# Meridian Flight - Setup Plan

Create a new Vue project called `meridian-flight`.

Use the create-vue scaffolding tool with Vite.

Enable:
- TypeScript
- Vue Router

Disable:
- JSX
- Pinia
- Vitest
- End-to-end testing
- ESLint
- Prettier

Then install and configure:
- Vuetify 3
- Material Design Icons

Set everything up, run the project locally, and confirm that the default application loads successfully.
